# 📅 Gerenciador de Rotina e Otimização Android

Este projeto foi desenvolvido por **Pablo Ruan** como parte dos meus estudos no curso de **Sistemas de Informação**. O objetivo é centralizar ferramentas de automação para Android e um gerenciador de rotina pessoal focado em saúde e organização.

## 🚀 Funcionalidades

- **Projeto Bulking:** Registro diário de peso com meta de alcançar os **57kg** e cálculo automático de hidratação.
- **Gestão de Treino:** Divisão de treino estruturada para 4 dias na semana.
- **Cuidados com Pet:** Sistema de lembretes e registros de cuidados para o meu gato, **Eren**.
- **Automação Android (ADB):** Comandos integrados para otimização de cache e ativação do servidor Brevent.
- **Roleta de Lazer:** Sorteio de atividades para momentos de descanso (jogos e animes).

## 🛠️ Tecnologias Utilizadas

- **Python 3:** Linguagem principal do script.
- **Git & GitHub:** Para controle de versão e hospedagem do código.
- **ADB (Android Debug Bridge):** Para comunicação direta com dispositivos móveis.
